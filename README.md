# EA2: Optimización del Proceso de Cobro en un Supermercado mediante Hilos y Concurrencia

**Curso:** Desarrollo de Software Seguro  
**Estudiante:** Daniel Andres Duque Alvarez  
**Institución:** IU Digital de Antioquia — 2026

## Descripción
Aplicación en Java que simula el proceso de cobro en un supermercado
usando concurrencia y programación multihilo.

## Clases
- `Producto` — Representa un artículo del supermercado
- `Cliente` — Representa un comprador con su lista de productos
- `Cajera` — Hilo que atiende clientes secuencialmente
- `Main` — Orquesta la simulación con ExecutorService

## Cómo ejecutar
1. Abre el proyecto en IntelliJ IDEA o Eclipse
2. Asegúrate de que los archivos estén en el paquete `supermercado`
3. Ejecuta `Main.java`
